POST entities/create

Description

Create a new entity. This is only used for custom entities added by users.

Details

  • OAuth Token Required

Parameters

title required The title of the custom entity.

subtitle required The subtitle of the custom entity. This should default to the category if no additional information is provided.

category required The category of the custom entity.

subcategory required The subcategory of the custom entity.

desc optional A brief description of the entity.

address optional The address of the entity.

coordinates optional The latitude and longitude of the entity, separated by a comma.

Sample Request

POST https://api.stamped.com/v1/entities/create.json
oauth_token=1234567890123456789012&title=Spotted+Pig&subtitle=New+York,+NY&category=food&subcategory=restaurant&desc=Burger+place&coordinates=40.714623,-74.006605

Sample Result

{
    "entity_id": "4e226ea84e61754161000000",
    "title": "Spotted Pig",
    "subtitle": "New York, NY"
    "category": "food",
    "subcategory": "restaurant"
    "desc": "Burger place",
    "coordinates": "40.714623,-74.006605",
    "last_modified": "2011-07-17 19:03:22.354813",
}

Description

Details

  • OAuth Token Required

POST entities/update

Description

Update details for a custom entity.

Details

  • OAuth Token Required
  • At least one field other than entity_id must be included. Any fields not included will not be changed. To remove a value for a field you can submit an empty string.
  • Users can only modify entities that they created.

Parameters

entity_id required The unique identifier of the entity being updated.

title optional The title of the custom entity.

subtitle optional The subtitle of the custom entity. This should default to the category if no additional information is provided.of the custom entity. If an invalid category is passed it will default to “Other”.

category optional The category of the custom entity. The default valud is “Other”.

subcategory optional The subcategory of the custom entity. The default valud is “Other”.

desc optional A brief description of the entity.

address optional The address of the entity.

coordinates optional The latitude and longitude of the entity, separated by a comma.

Sample Request

POST https://api.stamped.com/v1/entities/update.json
oauth_token=1234567890123456789012&entity_id=4e226ea84e61754161000000&desc=Gastropub+in+the+West+Village,+NYC

Sample Result

{
    "entity_id": "4e226ea84e61754161000000",
    "title": "Spotted Pig",
    "subtitle": "New York, NY"
    "category": "food",
    "subcategory": "restaurant"
    "desc": "Gastropub in the West Village, NYC",
    "coordinates": "40.714623,-74.006605",
    "last_modified": "2011-07-17 19:03:22.354813",
}

POST entities/remove

Description

Remove a custom entity. Warning: It is still undecided as to what effect this will have on any stamps that exist on the entity.

Details

  • OAuth Token Required
  • Only the creator of a custom entity can delete the entity.

Parameters

entity_id required The unique identifier of the entity being removed.

Sample Request

POST https://api.stamped.com/v1/entities/remove.json
oauth_token=1234567890123456789012&entity_id=4e226ea84e61754161000000

Sample Result

{

    "title": "Spotted Pig",

GET entities/search

Description

Search for an entity.

Details

  • OAuth Token Required
  • Note: Undecided if the distance calculation will happen on the backend or on the client.

Parameters

q required Text string to match against the entity title.

coordinates optional Coordinates of current location to prioritize local results. Coordinates should be in the format “lat,lng”.

Sample Request

GET https://api.stamped.com/v1/entities/search.json?oauth_token=1234567890123456789012&q=Spotted+Pig

Sample Result

[
    {
        "entity_id": "4e226ea84e61754161000000",
        "title": "Spotted Pig",
        "category": "Restaurant",
        "subtitle": "New York, NY",
    }
]
