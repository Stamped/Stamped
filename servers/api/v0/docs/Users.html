GET users/show

Description

Show detailed information for a given user.

Details

  • OAuth Token Required

Parameters

user_id or screen_name required The user id or screen name of the requested user. If the user id does not match the screen name then the user id will take precedence.

Sample Request

GET https://api.stamped.com/v1/users/show.json
oauth_token=1234567890123456789012&user_id=47cc67093475061e3d95369d

Sample Result

{
    "user_id": "47cc67093475061e3d95369d",
    "name": "Kevin Palms",
    "screen_name": "kevin",
    "bio": "The very purpose of existence is to reconcile the glowing opinion we have of ourselves with the appalling things that other people think about us.",
    "color_primary": "333333",
    "color_secondary": "999999",
    "privacy": false,
    "num_stamps": 0,
    "num_friends": 0,
    "num_followers": 0,
    "num_faves": 0,
    "num_credits": 0,
    "num_credits_given": 0
}

POST users/lookup

Description

Return detailed information for up to 100 users. Users can be specified by id, screen name, or a combination of the two.

Details

  • OAuth Token Required

Parameters

user_ids or screen_names required The user ids or screen names of the requested users. Multiple values should be separated by commas.

Sample Request

GET https://api.stamped.com/1/users/lookup.json
oauth_token=1234567890123456789012&user_ids=47cc67093475061e3d95369d,4e226fcd4e61754161000007&screen_names=kevin,robby

POST users/search

Description

Search users via their name, screen name, and bio. Returns a list of user objects, up to 20 at most.

Details

  • OAuth Token Required

Parameters

q required The search query to run.

limit optional The number of results to return. Default is 20.

Sample Request

GET https://api.stamped.com/1/users/search.json
oauth_token=1234567890123456789012&q=Kevin%20Palms

Sample Result

[
    {
        "user_id": "47cc67093475061e3d95369d",
        "name": "Kevin Palms",
        "screen_name": "kevin",

        "num_friends": 0,
        "num_followers": 0,
        "num_faves": 0,
        "num_credits": 0,
        "num_credits_given": 0

POST users/find/phone

Description

Details

Parameters
