USERS

GET users/show

Description

Shows detailed information for a given user.

Details

  • No Authentication

Parameters

user_id or screen_name required The user id or screen name of the requested user.

authenticated_user_id optional The user id of the authenticating user. This is necessary if the requested account is locked.

Sample Request

GET https://api.stamped.com/1/users/show.json
user_id=47cc67093475061e3d95369d

GET users/lookup

Description

Return detailed information for up to 100 users. Users can be specified by id, screen name, or a combination of the two.

Details

  • No Authentication

Parameters

user_ids or screen_names required The user ids or screen names of the requested users. Values should be separated by commas.

authenticated_user_id optional The user id of the authenticating user. This is necessary if any of the requested accounts are locked.

Sample Request

GET https://api.stamped.com/1/users/lookup.json
user_ids=47cc67093475061e3d95369d,4e226fcd4e61754161000007&screen_names=kevin,robby

GET users/search

Description

Search users via their name, screen name, and bio.

Details

  • No Authentication

Parameters

q required The search query to run.

limit optional The number of results to return. Default is 20.

Sample Request

GET https://api.stamped.com/1/users/search.json
q=Kevin%20Palms

Sample Result

[
    {
        "user_id": "47cc67093475061e3d95369d",
        "first_name": "Kevin",
        "last_name": "Palms",
        "screen_name": "kevin",
        "display_name": "Kevin P.",
        "bio": "The very purpose of existence is to reconcile the glowing opinion we have of ourselves with the appalling things that other people think about us.",
        "website": null,
        "color": {
            "primary": [91, 172, 253],
            "secondary": [95, 167, 56]
        }
    }
]

GET users/privacy

Description

Check whether a given user’s account is publicly available or not.

Details

  • No Authentication

Parameters

user_id or screen_name required The user id or screen name of the requested user.

Sample Request

GET https://api.stamped.com/1/users/privacy.json
user_id=47cc67093475061e3d95369d

Sample Result

false
