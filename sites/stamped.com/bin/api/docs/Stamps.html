STAMPS

POST stamps/create

Description

Create a new stamp for the authenticating user.

Details

  • Requires Authentication

Parameters

authenticated_user_id required The user id of the authenticating user.

entity_id required The unique id of the entity that the user is stamping.

blurb optional A brief text description of why the user is stamping the entity. This is currently limited to 512 characters.

image optional An image that can be attached to the stamp. Data must be base64-encoded, and the image must be a valid GIF, JPG, or PNG less than 700 kilobytes in size. Images larger than 500 pixels will be scaled down.

credit optional User information for any users that are named as receiving credit in the stamp. Multiple screen names must be separated by commas.

Sample Request

POST https://api.stamped.com/1/stamps/create.json
authenticated_user_id=47cc67093475061e3d95369d&entity_id=4e226fcd4e61754161000006&blurb=Best+place+ever!&credit=4e226fcd4e61754161000007,4e226fcd4e61754161000008

POST stamps/update

Description

Update details for a stamp owned by the authenticating user.

Details

  • Requires Authentication

Parameters

authenticated_user_id required The user id of the authenticating user. This user must be the owner of the stamp.

stamp_id required The unique id of the stamp.

blurb optional A brief text description of why the user is stamping the entity. This is currently limited to 512 characters.

image optional An image that can be attached to the stamp. Data must be base64-encoded, and the image must be a valid GIF, JPG, or PNG less than 700 kilobytes in size. Images larger than 500 pixels will be scaled down.

credit optional User information for any users that are named as receiving credit in the stamp. Multiple screen names must be separated by commas.

Sample Request

POST https://api.stamped.com/1/stamps/update.json
authenticated_user_id=47cc67093475061e3d95369d&stamp_id=4e226fcd4e18474561000000&blurb=Went+here+with+%40robby...+Really,+the+best+ever.

POST stamps/remove

Description

Remove a stamp owned by the authenticating user.

Details

  • Requires Authentication

Parameters

authenticated_user_id required The user id of the authenticating user. This user must be the owner of the stamp.

stamp_id required The unique id of the stamp.

Sample Request

POST https://api.stamped.com/1/stamps/remove.json
authenticated_user_id=47cc67093475061e3d95369d&stamp_id=4e226fcd4e18474561000000

Sample Result

true

GET stamps/show

Description

Show details for a given stamp.

Details

  • No Authentication

Parameters

stamp_id required The unique id of the stamp.

authenticated_user_id optional The user id of the authenticating user. This is necessary for stamps that are private.

GET https://api.stamped.com/1/stamps/show.json
