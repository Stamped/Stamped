POST stamps/create

Description

Create a new stamp for the authenticating user.

Details

  • OAuth Token Required

Parameters

entity_id required The unique id of the entity that the user is stamping.

blurb optional A brief text description of why the user is stamping the entity. This is currently limited to 512 characters. Any embedded mentions will be derived on the server.

image optional An image that can be attached to the stamp. Data must be base64-encoded, and the image must be a valid GIF, JPG, or PNG less than 700 kilobytes in size. Images larger than 500 pixels will be scaled down.

credit optional The screen names for any users that should receive credit for the stamp. Multiple screen names must be separated by commas.

Sample Request

POST https://api.stamped.com/v1/stamps/create.json
oauth_token=1234567890123456789012&entity_id=4e226ea84e61754161000000&blurb=Best+place+ever!&credit=robby,bart

POST stamps/update

Description

Update details for a stamp owned by the authenticating user.

Details

  • OAuth Token Required
  • At least one field other than stamp_id must be included. Any fields not included will not be changed. To remove a value for a field you can submit an empty string.
  • Users can only modify stamps that they created.

Parameters

stamp_id required The unique id of the stamp.

blurb optional A brief text description of why the user is stamping the entity. This is currently limited to 512 characters. Any embedded mentions will be derived on the server.

image optional An image that can be attached to the stamp. Data must be base64-encoded, and the image must be a valid GIF, JPG, or PNG less than 700 kilobytes in size. Images larger than 500 pixels will be scaled down.

credit optional The screen names for any users that should receive credit for the stamp. Multiple screen names must be separated by commas.

Sample Request

POST https://api.stamped.com/v1/stamps/update.json
oauth_token=1234567890123456789012&&stamp_id=4e226fcd4e18474561000000&blurb=Went+here+with+%40robby...+Really,+the+best+ever.
