POST favorites/create

Description

Add a given entity to the authenticated user’s list. If inspired by a stamp, the stamp’s unique id is recorded as well.

Details

  • OAuth Token Required

Parameters

entity_id required The id of the entity added to the list

stamp_id optional The id of the stamp that the entity was saved from. This is used to suggest credit upon completion of the item.

Sample Request

POST https://api.stamped.com/v1/favorites/create.json
oauth_token=1234567890123456789012&entity_id=4e26294e32a7ba0d4a000005

GET favorites/show

Description

Return all favorites for a given user.

Details

  • OAuth Token Required

Parameters

limit optional The maximum number of desired results. Values greater than the default (20) will be ignored.

since optional A UNIX timestamp stating the beginning of the time slice desired. This enables a certain range of stamps to be returned.

before optional A UNIX timestamp stating the end of the time slice desired. This enables a certain range of stamps to be returned.

Sample Request

GET https://api.stamped.com/v1/favorites/show.json?oauth_token=1234567890123456789012

POST favorites/remove

Description

Remove a given item from the list of favorites. This does not destroy any stamps that were created from the item.

Details

  • OAuth Token Required

Parameters

entity_id required The unique id of the favorited entity.

Sample Request

POST https://api.stamped.com/v1/favorites/remove.json
